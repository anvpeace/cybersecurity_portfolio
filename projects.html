Student Management Menu
GitHub Repo

This project leverages abstract base classes, inheritance, templates, file I/O, dynamic memory management, and operator overloading to:

  • Store and manage `Student` and `Faculty` records using a polymorphic linked list structure
  • Input data from the console, text files, or CSV files to dynamically create and insert objects
  • Search for records by ID or name and remove them from the list
  • Sort records by ID, name, or GPA using comparison-based algorithms
  • Format output neatly with overloaded stream insertion operators for both 'Student' and 'Faculty' objects

Prime Number Overload Operators
GitHub Repo

This project leverages constructors, mutators, member & non-member fucntions, and a comprehensive set of overload operators to:

  • Determine whether a given integer is prime
  • Step forward or backward through the sequence of prime numbers
  • Perform arithmetic “jumps” of 'n' primes
  • Access individual digits of a number with the subscript operator
